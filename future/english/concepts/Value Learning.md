# Value Learning

## English Abstract

Value learning is the proposal that an AI should learn what humans value from evidence and a criterion, rather than receiving a fully explicit final objective from the start.

## Current Synthesis

In the current corpus value learning is one of the main alternatives to brute-force reward optimization. It keeps the final target abstract while letting beliefs about the target improve over time.

## Supporting Evidence

- [[english/sources/2014-nick-bostrom-superintelligence#acquiring values]]

## Disagreements And Uncertainties

- The central difficulty is not "learning" in the ordinary ML sense but defining the right evidence criterion and reference target.
- Many later alignment disputes can be read as arguments over whether present systems have anything like this.
