# Coherent Extrapolated Volition

## English Abstract

Coherent extrapolated volition is an indirect value-loading idea associated with early alignment thinking: instead of specifying exact goals, aim at what humanity would collectively want under better reflection and understanding.

## Current Synthesis

In the current corpus, CEV matters less as an implementation recipe and more as evidence that value specification has long been understood as an indirect, philosophically difficult problem.

## Disagreements And Uncertainties

- The corpus does not contain a concrete modern implementation path for CEV.
- It remains useful mainly as a conceptual marker for how hard human-value targeting really is.
