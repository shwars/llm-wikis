# GitHub

## English Abstract

Platform that helped mainstream AI pair programming and then background coding agents inside repo workflows.

## Current Synthesis

GitHub's arc in AI-Assisted Software Development runs from inline pair programming with Copilot to asynchronous coding agents and terminal-native Copilot CLI. Its distinctive advantage is the direct embedding of delegation and review into the repository and pull-request workflow.
