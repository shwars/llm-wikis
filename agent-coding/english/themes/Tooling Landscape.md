# Tooling Landscape

## English Abstract

This theme summarizes the main product categories and competitive directions in AI-Assisted Software Development as of April 22, 2026.

## Current Synthesis

The market now spans at least seven overlapping clusters. [[english/tools/GitHub Copilot|Code-completion tools]] keep the human fully in control. [[english/tools/Aider|Prompt-review-edit tools]] and [[english/tools/Gemini Code Assist|IDE copilots]] add conversational editing and reviewable transformations. [[english/tools/Claude Code|Terminal agents]] such as [[english/tools/Claude Code|Claude Code]], [[english/tools/OpenCode|OpenCode]], [[english/tools/Qwen Code|Qwen Code]], and [[english/tools/Gemini CLI|Gemini CLI]] emphasize scriptability and repo-local control. [[english/tools/Cursor|IDE agents]] such as [[english/tools/Cursor|Cursor]] and [[english/tools/Windsurf|Windsurf]] fold those loops back into the editor. [[english/tools/Codex|Repo-native and background agents]] like [[english/tools/Codex|Codex]], [[english/tools/OpenHands|OpenHands]], [[english/tools/Devin|Devin]], and [[english/tools/GitHub Copilot Coding Agent|GitHub Copilot Coding Agent]] shift supervision toward tasks, environments, and PRs. Finally, [[english/tools/Stitch|design-side agents]] and [[english/tools/Claude Design|artifact-generation tools]] increasingly shape what implementation agents do next.

## Cluster Guide

- `Code completion`: [[english/tools/GitHub Copilot|GitHub Copilot]] and the inline side of [[english/tools/GigaCode|GigaCode]].
- `Prompt-review-edit`: [[english/tools/Aider|Aider]], [[english/tools/Gemini Code Assist|Gemini Code Assist]], and the chat side of [[english/tools/GigaCode|GigaCode]].
- `CLI agentic development`: [[english/tools/Claude Code|Claude Code]], [[english/tools/Gemini CLI|Gemini CLI]], [[english/tools/OpenCode|OpenCode]], [[english/tools/Qwen Code|Qwen Code]], and [[english/tools/GitHub Copilot CLI|GitHub Copilot CLI]].
- `IDE agentic development`: [[english/tools/Cursor|Cursor]], [[english/tools/Windsurf|Windsurf]], and parts of [[english/tools/Antigravity|Antigravity]].
- `Repo or background delegation`: [[english/tools/Codex|Codex]], [[english/tools/OpenHands|OpenHands]], [[english/tools/Devin|Devin]], and [[english/tools/GitHub Copilot Coding Agent|GitHub Copilot Coding Agent]].
- `Design-adjacent workflow tools`: [[english/tools/Stitch|Stitch]] and [[english/tools/Claude Design|Claude Design]].
- `General-use overlap`: [[english/tools/Claude Desktop|Claude Desktop]] plus the broader cluster mapped in [[english/analyses/General Use Agents - 2026|General Use Agents - 2026]].
