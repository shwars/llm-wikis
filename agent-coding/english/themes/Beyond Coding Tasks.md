# Beyond Coding Tasks

## English Abstract

This theme tracks where coding-agent harnesses are already being used for adjacent technical and knowledge-work loops.

## Current Synthesis

A major 2025-2026 shift is that the same tools and harnesses used for coding are now used for deep research, documentation, note-taking, issue triage, release briefs, CI summaries, visual QA, spreadsheet work, and inbox-like follow-up. The common denominator is not code output itself, but the ability to maintain context, call tools, verify results, and carry work forward over time. The rise of general-use agents like OpenClaw, Hermes Agent, Memoh, and Goose makes this explicit: coding is becoming one strong workflow inside a broader agent stack rather than the only destination.
