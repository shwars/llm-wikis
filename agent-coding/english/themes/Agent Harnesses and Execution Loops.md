# Agent Harnesses and Execution Loops

## English Abstract

This theme focuses on the scaffolding that turns a raw model into a reliable software worker.

## Current Synthesis

The modern AI-Assisted Software Development harness is a system, not a prompt. It includes repo instructions, task decomposition, tool permissions, worktree isolation, evaluator stages, feedback loops, and supervision interfaces. Across Anthropic, OpenAI, GitHub, and Google sources, the winning pattern is consistent: keep the loop simple enough to debug, but rich enough to verify and recover. MCP and Agent Skills now sit inside this harness picture too: MCP brings live external systems into the loop, while skills package reusable procedures that keep the loop legible and cheap.
