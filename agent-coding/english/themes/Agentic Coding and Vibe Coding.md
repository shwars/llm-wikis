# Agentic Coding and Vibe Coding

## English Abstract

This theme distinguishes the broad AI-Assisted Software Development field from the narrower meme and style called vibe coding.

## Current Synthesis

Agentic coding is the broad umbrella: systems that can explore, plan, edit, run commands, verify, and sometimes work asynchronously. Vibe coding is a narrower cultural style inside that umbrella, centered on fast natural-language prototyping, heavy trust in generation, and lighter direct code inspection. The main practical distinction is not seriousness versus unseriousness, but verification depth and expected maintenance cost.
