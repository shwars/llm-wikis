# GigaCode Inline Code Assistant

## Summary

The inline-assistant docs anchor [[english/tools/GigaCode|GigaCode]] in the code-completion lineage. The product supports whole-line suggestions, full-function suggestions, and comment-to-code completions in IDEs, while explicitly telling users to switch to CodeChat for broader reasoning tasks.

## Key Claims

- GigaCode provides inline completions for both VS Code and JetBrains.
- The product spans single-line, multi-line, and comment-driven generation.
- The docs explicitly separate inline completion from broader chat-based assistance.

## Notable Practices

- Keep completion and chat workflows separate when their task shapes differ.
- Use comment-to-code flows for low-friction intent capture inside the editor.
- Treat IDE completion as a distinct maturity layer from repo-operating agents.
