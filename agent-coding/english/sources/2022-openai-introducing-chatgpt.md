# Introducing ChatGPT

## Summary

ChatGPT changed coding workflows by making conversational problem solving mainstream. After November 30, 2022, many developers stopped using LLMs only for completion and started using them for debugging, explanation, planning, and code generation through iterative dialogue.

## Key Claims

- Dialogue itself became a useful programming interface.
- Coding assistance expanded from local code suggestions to general reasoning and troubleshooting.
- The model-user loop shifted from accept/reject to iterative clarification.

## Notable Practices

- Use conversational tools for scoping, explanation, and debugging before implementation.
- Refine tasks through follow-up turns instead of assuming the first answer is final.
- Keep error messages and concrete examples in the loop.
