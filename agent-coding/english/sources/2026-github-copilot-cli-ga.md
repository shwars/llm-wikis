# GitHub Copilot CLI is now generally available

## Summary

Copilot CLI signals GitHub's move into terminal-native AI-Assisted Software Development. The framing is notable: the CLI is no longer just a shell helper but an agentic environment with planning, memory, execution, and variable autonomy settings.

## Key Claims

- The terminal has become a primary surface for professional agentic coding.
- Persistent memory and autonomy controls are now standard expectations.
- GitHub is converging with other terminal-agent products.

## Notable Practices

- Use terminal agents when you need scriptability and repo-local control.
- Tune autonomy level to task risk and verification strength.
- Treat session memory as a workflow feature, not just a convenience.
