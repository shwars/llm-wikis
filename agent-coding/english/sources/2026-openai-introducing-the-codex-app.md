# Introducing the Codex app

## Summary

The Codex app makes coordination itself a product surface: threads, projects, worktrees, skills, and automations are all first-class. AI-Assisted Software Development here is no longer only about generating patches, but about supervising teams of agents over time across coding and adjacent knowledge-work tasks.

## Key Claims

- Multi-agent coordination needs dedicated UX, not only a better model.
- Skills and automations expand coding agents into broader workflows.
- Worktrees are core infrastructure for parallelism.

## Notable Practices

- Use app- or platform-level coordination for long-running parallel work.
- Package repeated workflows as skills and recurring automations.
- Delegate non-coding maintenance tasks into the same agent stack.
