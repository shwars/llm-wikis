# Building agents with the Claude Agent SDK

## Summary

Anthropic's Agent SDK post makes a key conceptual move: the useful object is not just a coding model but a reusable agent harness. The same loop can power deep research, note-taking, and other knowledge work, which broadens AI-Assisted Software Development from 'AI writes code' to 'software-shaped agents perform technical work.'

## Key Claims

- The harness pattern generalizes beyond coding.
- Agent loops benefit from reusable skills, memory, and supervision interfaces.
- Coding agents are early examples of a broader class of software-shaped agents.

## Notable Practices

- Design agent harnesses as reusable infrastructure, not one-off prompts.
- Move stable workflows into skills and structured tools.
- Look for adjacent non-coding tasks that share the same loop shape.
