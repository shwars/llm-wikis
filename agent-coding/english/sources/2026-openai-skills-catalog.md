# OpenAI skills catalog snapshot

## Summary

The OpenAI skills catalog turns the skill idea into a shared distribution surface: reusable folders of instructions, resources, and scripts for repeatable tasks. It matters because it shows how skills can become an ecosystem rather than only a local customization trick.

## Structure Map

- `[[#Summary]]`: what the skills catalog is and why it matters.

## Key Claims

- Skills are increasingly treated as portable building blocks across agent products.
- Catalog and installer workflows make procedural knowledge distributable.
- The same substrate can serve built-in, curated, and experimental workflows.

## Notable Practices

- Package recurring team workflows as installable skill folders.
- Separate stable skills from experimental ones in distribution.
- Use catalogs to make organizational know-how searchable and shareable.
