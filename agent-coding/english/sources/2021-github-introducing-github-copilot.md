# Introducing GitHub Copilot: your AI pair programmer

## Summary

GitHub presented Copilot as an AI pair programmer that reads local code context and suggests lines or functions inline. In the AI-Assisted Software Development history, this is the code-completion era: the human still decomposes tasks and owns verification, while the model accelerates implementation at the cursor.

## Key Claims

- Nearby code and comments became a usable prompt substrate for code generation.
- Testing and API exploration were already visible as early high-value use cases.
- The interaction model was assistive rather than autonomous.

## Notable Practices

- Keep a clean local code context; naming and comments feed the model.
- Treat suggestions as accelerators, not proofs of correctness.
- Use completion tools for narrow implementation and recall tasks.
