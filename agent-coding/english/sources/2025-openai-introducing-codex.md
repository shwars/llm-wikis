# Introducing Codex

## Summary

OpenAI's 2025 Codex launch marks the transition from interactive coding help to cloud-executed software agents. Separate sandboxes, task parallelism, AGENTS.md guidance, and verifiable logs move the workflow closer to asynchronous collaboration with junior engineers or contractor threads.

## Key Claims

- Cloud agents can work on multiple tasks independently in parallel.
- AGENTS.md is a concrete interface between repo owners and the agent.
- Evidence such as terminal logs and test output matters for trust.

## Notable Practices

- Scope tasks tightly and run several agents in parallel.
- Give the agent stable environment instructions inside the repo.
- Demand verifiable outputs, not just plausible explanations.
