# OpenHands introduction snapshot

## Summary

OpenHands represents the platformized open-source end of AI-Assisted Software Development: not just a coding assistant, but a software-agent stack with cloud, CLI, SDK, MCP, and custom sandbox support. It is especially relevant for teams that want to self-host or deeply customize the agent harness.

## Key Claims

- Open-source teams are building full agent platforms, not just single tools.
- SDK plus sandbox control matters for advanced deployment.
- Customization and self-hosting remain important parts of the landscape.

## Notable Practices

- Use platform tools when you need custom sandboxes and integrations.
- Package repo setup as scripts and microagents.
- Treat open-source agent stacks as infrastructure choices, not only UI choices.
