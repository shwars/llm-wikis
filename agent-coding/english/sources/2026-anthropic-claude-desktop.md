# Install Claude Desktop

## Summary

Anthropic's help-center docs show that [[english/tools/Claude Desktop|Claude Desktop]] is no longer just a chat wrapper. With extensions, local [[english/concepts/Model Context Protocol|MCP]], and Cowork, it becomes a visual surface for agentic work that overlaps with coding, research, and desktop automation.

## Key Claims

- Claude Desktop supports local extensions and curated extension discovery.
- Cowork brings Claude Code-like multi-step execution into a visual desktop workflow.
- Parallel workstreams and isolated execution are now part of the desktop product story.

## Notable Practices

- Use desktop agents when visual supervision matters more than shell-native composition.
- Treat extensions and MCP access as security boundaries that need review.
- Use Cowork-style execution for long-running tasks that benefit from a visible review surface.
