# Analyzing coding agent transcripts to upper bound productivity gains

## Summary

METR's transcript study suggests that coding agents can produce real productivity gains, but the magnitude is highly task- and measurement-dependent. The note is useful less as a headline number than as a reminder that agentic productivity claims need careful interpretation and often overstate full-workflow uplift.

## Key Claims

- Productivity gains appear real but hard to measure cleanly.
- Task selection biases the observed uplift upward.
- Transcript analysis is useful, but it does not replace full human-uplift studies.

## Notable Practices

- Use verification and task slicing if you want measurable wins.
- Separate local session speed from overall organizational productivity.
- Be skeptical of universal x-times-faster slogans.
