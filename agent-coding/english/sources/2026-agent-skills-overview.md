# Agent Skills overview

## Summary

The Agent Skills overview is the standards-layer statement of the skill idea: portable folders of instructions, scripts, and resources that agents can discover on demand. It matters because it lifts the skill concept out of any one vendor interface.

## Structure Map

- `[[#Summary]]`: the standards-layer definition and why it matters.

## Key Claims

- Skills are a portable knowledge-and-procedure package, not just a chat prompt.
- The same format can capture team, company, and user-specific workflows.
- Open skill standards let multiple agent products share the same procedural assets.

## Notable Practices

- Treat high-value procedures as versioned assets rather than copy-pasted chat fragments.
- Use open skill formats when you want portability across vendors.
- Keep the skill package small at the entrypoint and richer in referenced assets.
