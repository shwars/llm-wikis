# Windsurf docs snapshot

## Summary

Windsurf packages many now-familiar AI-Assisted Software Development primitives into an IDE-first environment: agent chat, memories, rules, worktrees, workflows, AGENTS.md, and deploy hooks. It is a good example of how quickly terminal-agent ideas have been absorbed back into graphical IDE products.

## Key Claims

- IDE products now expose many of the same primitives as terminal agents.
- Memory, rules, AGENTS.md, and worktrees are becoming common vocabulary.
- Deployment and workflow automation are moving into the editor surface.

## Notable Practices

- Use IDE-centric tools when team adoption depends on familiar GUI affordances.
- Keep memories and rules explicit and reviewable.
- Treat deploy workflows as part of the agent harness.
