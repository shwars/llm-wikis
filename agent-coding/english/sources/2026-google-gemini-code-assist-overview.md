# Gemini Code Assist overview

## Summary

Google's official overview places [[english/tools/Gemini Code Assist|Gemini Code Assist]] in the IDE-first branch of AI-assisted development. It combines code completion, chat, code generation, debugging help, and source citations, while also pointing toward agent mode as the next step up the maturity ladder.

## Key Claims

- Gemini Code Assist spans free, Standard, and Enterprise editions.
- The product is IDE-native, citation-aware, and broader than raw autocomplete.
- Agent mode is an explicit continuation path inside the same product family.

## Notable Practices

- Treat source citations as a review aid rather than a substitute for verification.
- Use IDE-first assistants when teams want a lower-friction path from completion to conversational editing.
- Expect IDE assistants to blur into agent mode over time.
