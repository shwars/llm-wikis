# Copilot coding agent is now generally available

## Summary

GitHub's coding agent brings AI-Assisted Software Development directly into the repository workflow: issues become delegations, background environments produce draft PRs, and review happens through existing GitHub surfaces. It is less terminal-centric than Claude Code or Codex, but stronger on repo-native task routing and PR-based supervision.

## Key Claims

- Background agents can fit naturally into issue and PR workflows.
- Repo-native orchestration changes who can delegate work, not just how code is written.
- Asynchronous review is a core part of the interface.

## Notable Practices

- Use issue granularity and PR review as control surfaces.
- Favor isolated, reviewable tasks for background agents.
- Let the repo workflow carry supervision metadata.
