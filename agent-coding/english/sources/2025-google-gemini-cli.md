# Gemini CLI: your open-source AI agent

## Summary

Gemini CLI frames the terminal agent as a general local utility rather than a coding-only assistant. Its combination of open source, scriptability, web grounding, MCP, and large context makes it relevant both for AI-Assisted Software Development and for adjacent research or operations tasks.

## Key Claims

- Terminal agents can unify coding and non-coding workflows.
- Openness and scriptability are core differentiators in CLI tools.
- Context size and tool use are now product-level selling points.

## Notable Practices

- Use CLI agents for mixed automation, scripting, and research flows.
- Exploit non-interactive mode for repeatable tasks.
- Use MCP and search grounding when freshness matters.
