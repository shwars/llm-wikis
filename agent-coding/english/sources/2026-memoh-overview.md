# Memoh overview readme snapshot

## Summary

Memoh shows how the always-on agent pattern can turn into a multi-bot operations platform: container isolation, GUI administration, channels, MCP, and skills. It matters because it makes the boundary between general assistant systems and agent infrastructure increasingly thin.

## Key Claims

- General-use agents can be productized as multi-bot infrastructure, not just one personal assistant.
- Isolation and admin UX are becoming central differentiators in long-running agent systems.
- MCP and skills are now expected even outside coding-first products.

## Notable Practices

- Use container boundaries when long-running agents need strong operational separation.
- Give operators visible configuration surfaces for channels, MCP, skills, and models.
- Treat memory and filesystem isolation as part of the system contract.
