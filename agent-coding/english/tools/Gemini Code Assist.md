# Gemini Code Assist

## English Abstract

Google's IDE-first coding assistant that blends code completion, chat, citations, and agent-mode progression.

## Current Synthesis

[[english/tools/Gemini Code Assist|Gemini Code Assist]] shows how modern IDE assistants increasingly bundle three layers together: completion, conversational editing, and a pathway into agent mode. It is best understood as Google's bridge between classic IDE copilots and more autonomous terminal or platform agents.

## Tool Profile

- `Primary surface`: IDE.
- `Maturity stage`: Prompt-review-edit workflow.
- `Autonomy model`: assistive to semi-agentic, with documented movement toward agent mode.
- `Strengths`: IDE integration, chat, citations, and low-friction adoption for teams already living in editors.
- `Limits`: less autonomous by default than terminal-first or repo-first agent systems.
