# Windsurf

## English Abstract

An IDE-native agentic development environment organized around memories, rules, worktrees, workflows, and deploy hooks.

## Current Synthesis

[[english/tools/Windsurf|Windsurf]] shows how many terminal-agent ideas have already been absorbed back into graphical IDE products. It combines editor convenience with memory, rules, worktrees, and workflow automation, making it a strong example of agentic development in an IDE shell.

## Tool Profile

- `Primary surface`: IDE.
- `Maturity stage`: Agentic development.
- `Autonomy model`: editor-native agent workflows with memory and workflow infrastructure.
- `Strengths`: IDE ergonomics, persistent rules, worktrees, and deployment-oriented flows.
- `Limits`: less shell-composable than terminal-first systems.
