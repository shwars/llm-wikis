# Gemini CLI

## English Abstract

A terminal agent from Google that spans coding, research, and automation with MCP and web-grounded workflows.

## Current Synthesis

[[english/tools/Gemini CLI|Gemini CLI]] matters because it treats the terminal agent as a general local utility rather than a coding-only assistant. It sits in the same family as [[english/tools/Claude Code|Claude Code]] and [[english/tools/OpenCode|OpenCode]], but with a stronger bridge to Google's broader assistant and IDE ecosystem.

## Tool Profile

- `Primary surface`: CLI / terminal.
- `Maturity stage`: Agentic development.
- `Autonomy model`: interactive terminal agent with tool use and scriptability.
- `Strengths`: open-source posture, web grounding, MCP, and flexible non-coding utility.
- `Limits`: weaker repo-native delegation surfaces than worktree-first or PR-first systems.
