# Claude Desktop

## English Abstract

A visual desktop surface for Claude that now overlaps with coding agents through extensions, local MCP, and Cowork.

## Current Synthesis

[[english/tools/Claude Desktop|Claude Desktop]] sits between a general-purpose desktop agent and a visual coding supervisor. Anthropic's current desktop story matters here because Cowork brings [[english/tools/Claude Code|Claude Code]]-like multi-step execution into an app surface with local files, extensions, and controlled execution.

## Tool Profile

- `Primary surface`: desktop / visual workspace.
- `Maturity stage`: Multi-agent development.
- `Autonomy model`: visual agent supervision with long-running parallel workstreams.
- `Strengths`: local-file access, extensions, MCP, and a friendlier review surface than a raw shell.
- `Limits`: less composable and less terminal-native than CLI-first coding agents.
