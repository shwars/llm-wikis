# GitHub Copilot

## English Abstract

The iconic IDE pair-programmer that defined the modern code-completion era of AI-assisted software development.

## Current Synthesis

[[english/tools/GitHub Copilot|GitHub Copilot]] is the clearest anchor for the code-completion stage in this wiki's historical model. Even as the GitHub ecosystem expanded into CLI and background agents, the original product remains the reference point for inline assistive coding.

## Tool Profile

- `Primary surface`: IDE.
- `Maturity stage`: Code completion.
- `Autonomy model`: low-autonomy assistive completion and chat.
- `Strengths`: low friction, strong inline ergonomics, historic influence on the field.
- `Limits`: much weaker task delegation than later CLI and repo-native agents.
