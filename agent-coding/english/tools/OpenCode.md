# OpenCode

## English Abstract

An open-source terminal coding agent that emphasizes privacy, provider choice, and repo-local instructions.

## Current Synthesis

[[english/tools/OpenCode|OpenCode]] is important as evidence that the Claude Code / Codex workflow is already being replicated in the open. It reinforces the view that repo-local instructions and terminal-first agent loops are becoming standard interfaces rather than single-vendor quirks.

## Tool Profile

- `Primary surface`: CLI / terminal.
- `Maturity stage`: Agentic development.
- `Autonomy model`: interactive coding agent with multi-session and provider-flexible workflows.
- `Strengths`: open source, inspectability, privacy-first posture, AGENTS.md-friendly workflow.
- `Limits`: less polished than flagship closed products in some supervision surfaces.
