# GitHub Copilot Coding Agent

## English Abstract

A repo-native background agent that turns GitHub issues and tasks into delegated implementation work.

## Current Synthesis

[[english/tools/GitHub Copilot Coding Agent|GitHub Copilot Coding Agent]] matters because it moves the GitHub product line from assistance inside an editor to delegation inside the repository itself. The main boundary shifts from prompt and cursor to issue, branch, environment, and pull request.

## Tool Profile

- `Primary surface`: repo.
- `Maturity stage`: Agentic development.
- `Autonomy model`: background task execution routed through GitHub surfaces.
- `Strengths`: issue-to-PR workflow, repo-native supervision, familiar review surface.
- `Limits`: narrower local control than terminal-first agents and less open customization than self-hosted stacks.
