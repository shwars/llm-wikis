# Cursor

## English Abstract

An IDE-first agentic coding environment that helped normalize natural-language editing and background-agent workflows.

## Current Synthesis

[[english/tools/Cursor|Cursor]] is one of the clearest bridges from conversational editing to full agent mode inside the editor. It matters historically because it helped move teams from prompt-review-edit habits toward more autonomous IDE-supervised development.

## Tool Profile

- `Primary surface`: IDE.
- `Maturity stage`: Agentic development.
- `Autonomy model`: editor-native agent execution with optional background work.
- `Strengths`: tight editing loop, discoverable UI, strong diff-and-review ergonomics.
- `Limits`: still more editor-centric than repo-native or orchestration-first systems.
