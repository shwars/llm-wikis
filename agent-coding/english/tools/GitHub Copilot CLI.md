# GitHub Copilot CLI

## English Abstract

A terminal-native GitHub companion that brings Copilot-style assistance into command-line workflows.

## Current Synthesis

[[english/tools/GitHub Copilot CLI|GitHub Copilot CLI]] shows how completion-era brands are being reworked into terminal agents. It is closer to interactive terminal assistance than to fully delegated repo execution, but it clearly moves the Copilot family beyond editor-only help.

## Tool Profile

- `Primary surface`: CLI / terminal.
- `Maturity stage`: Agentic development.
- `Autonomy model`: interactive terminal agent inside the GitHub ecosystem.
- `Strengths`: GitHub integration, shell proximity, lower friction for teams already using Copilot.
- `Limits`: less orchestration-heavy than dedicated background-agent platforms.
