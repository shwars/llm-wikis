# Aider

## English Abstract

A terminal pair-programming tool that sits close to the prompt-review-edit end of the coding-agent spectrum.

## Current Synthesis

[[english/tools/Aider|Aider]] is important because it shows how much value still lives below full autonomy. It combines codebase-aware chat, git-friendly editing, and human review discipline without trying to become a long-running delegated engineer.

## Tool Profile

- `Primary surface`: CLI / terminal.
- `Maturity stage`: Prompt-review-edit workflow.
- `Autonomy model`: interactive pair-programming with low to medium autonomy.
- `Strengths`: git ergonomics, explicit human review, low operational overhead.
- `Limits`: weaker background delegation and weaker multi-agent orchestration than frontier agent stacks.
