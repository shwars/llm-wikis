# Antigravity

## English Abstract

Google's agent-first development platform built around editor, terminal, browser, and manager-style orchestration surfaces.

## Current Synthesis

[[english/tools/Antigravity|Antigravity]] represents the clearest Google-native move from IDE assistance toward orchestration of multiple agents and verifiable artifacts. In this wiki it marks the transition from editor help to platform-level agent supervision.

## Tool Profile

- `Primary surface`: IDE / platform.
- `Maturity stage`: Multi-agent development.
- `Autonomy model`: high-autonomy agents with manager-style orchestration.
- `Strengths`: agent-first UX, cross-surface execution, artifact-based trust building.
- `Limits`: platform complexity is higher than simpler editor copilots or CLI tools.
