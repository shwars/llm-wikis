# Claude Design

## English Abstract

A design-adjacent agentic workspace for prototypes, slides, and visual artifacts that feed software workflows.

## Current Synthesis

[[english/tools/Claude Design|Claude Design]] is not a code editor, but it belongs in this catalog because it turns visual intent into machine-usable artifacts that can shape implementation. In this wiki it sits on the boundary between design generation and coding workflow preparation.

## Tool Profile

- `Primary surface`: visual workspace.
- `Maturity stage`: Prompt-review-edit workflow.
- `Autonomy model`: visual generation and iterative refinement with human review.
- `Strengths`: rapid prototyping, artifact generation, design-to-implementation handoff.
- `Limits`: less suited to direct repo operations than coding-native agents.
