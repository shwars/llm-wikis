# Codex

## English Abstract

OpenAI's coding-agent system spanning cloud execution, app supervision, worktrees, skills, and automations.

## Current Synthesis

[[english/tools/Codex|Codex]] is one of the strongest examples of how coding tools have become orchestration systems. The shift from the 2025 launch to the 2026 app and harness-engineering material makes [[english/tools/Codex|Codex]] central to the move from agentic editing toward multi-agent, project-level supervision.

## Tool Profile

- `Primary surface`: repo / app / cloud.
- `Maturity stage`: Multi-agent development.
- `Autonomy model`: delegated background execution across isolated environments and projects.
- `Strengths`: worktrees, skills, automations, project memory, and strong supervision surfaces.
- `Limits`: requires disciplined task slicing and review to stay reliable.
