# OpenHands

## English Abstract

An open-source platform for agent-driven software development across cloud, CLI, SDK, and custom sandbox surfaces.

## Current Synthesis

[[english/tools/OpenHands|OpenHands]] matters because it packages agentic development as a platform, not just an assistant. It is especially relevant to teams that want self-hosting, custom sandboxes, SDK control, or deeper integration with their own harness.

## Tool Profile

- `Primary surface`: repo / cloud / SDK.
- `Maturity stage`: Multi-agent development.
- `Autonomy model`: high-autonomy delegated development on an extensible platform.
- `Strengths`: open-source stack, self-hosting, SDK access, and customizable execution environments.
- `Limits`: operational overhead is higher than for turnkey IDE or CLI products.
