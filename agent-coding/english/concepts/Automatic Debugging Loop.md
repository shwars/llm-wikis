# Automatic Debugging Loop

## English Abstract

The iterative cycle of implement, execute, observe, patch, and rerun until an explicit check passes or uncertainty is escalated.

## Current Synthesis

This loop is the operational heart of mature AI-Assisted Software Development. The model does not merely emit code; it tries code against reality. The more cheaply the system can execute and inspect failures, the closer agentic coding gets to autonomous software construction.
