# AI-Assisted Software Development

## English Abstract

An umbrella term for using language models to help create, change, test, understand, and maintain software.

## Current Synthesis

AI-Assisted Software Development is broader than chat-based coding help and broader than vibe coding. It covers the full spectrum from completion and pair programming to autonomous background agents and repository-shaped harnesses. The field is best understood through components such as problem specification, context management, agent management, and verification loops.
