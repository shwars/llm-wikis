# Computer Use

## English Abstract

The ability of an agent to see, click, type, and navigate graphical interfaces as part of task execution.

## Current Synthesis

Computer use extends AI-Assisted Software Development beyond source files and shells. Once an agent can operate browsers, apps, and desktop workflows, software creation overlaps with QA, deployment, research, and operations. This expands leverage but also increases safety, supervision, and reliability demands. It also explains the convergence between coding agents and general-use agents: desktop generalists and always-on personal agents increasingly need the same click, read, type, and verification loops.
