# Subagents

## English Abstract

Focused secondary agents used to preserve context and parallelize bounded subtasks.

## Current Synthesis

Subagents allow the main agent to stay focused while bounded side investigations or implementations happen elsewhere. They reduce context pollution, make responsibilities clearer, and support orchestrator-worker patterns in both coding and research tasks.
