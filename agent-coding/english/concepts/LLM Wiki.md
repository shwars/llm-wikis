# LLM Wiki

## English Abstract

A maintained markdown knowledge base that an LLM or agent can compile, update, navigate, and query over time.

## Current Synthesis

The LLM Wiki pattern separates raw material from synthesized notes and makes the note layer agent-maintained. It turns file structure, indexes, backlinks, and concept pages into context infrastructure rather than passive documentation.
