# Context Engineering

## English Abstract

The practice of curating the smallest durable high-signal working state for an agent.

## Current Synthesis

Context engineering treats tokens as scarce operating capital. The job is to decide what the agent should know now, what should live in durable files, what should be fetched just in time, what should be summarized away, and when another agent should take over part of the search space.
