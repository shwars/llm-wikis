# Agent Skills

## English Abstract

Portable folders of instructions, resources, scripts, and metadata that agents can discover and load on demand as reusable procedural knowledge.

## Current Synthesis

Agent Skills are procedural memory packages for agents. Unlike a root instruction file that loads every session, a skill usually activates only when it is relevant. That keeps the hot context small while still making large playbooks, helper scripts, examples, and policies available when needed. Skills can be personal, project-local, workspace-local, or catalog-installed, and increasingly follow open formats that work across products.
