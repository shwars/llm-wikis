# Agent Legibility

## English Abstract

The degree to which an agent's reasoning, actions, evidence, and uncertainty remain inspectable by humans.

## Current Synthesis

Legibility is what keeps higher autonomy from turning into blind trust. Logs, diffs, explicit plans, verification artifacts, citations, and uncertainty reports are all ways of making an agent reviewable. In AI-Assisted Software Development, stronger legibility often matters as much as stronger raw capability.
