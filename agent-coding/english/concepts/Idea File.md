# Idea File

## English Abstract

A structured problem-description artifact meant to be reused by agents, not just read by humans.

## Current Synthesis

The idea file sits between a prompt, a spec, and a project brief. It captures the intended artifact at a level abstract enough to be portable across models and tools, but concrete enough to let an agent instantiate a high-quality first version without rediscovering the problem from scratch.
