# Install Claude Desktop

## Captured Text

- Claude Desktop is the local desktop application for Claude on macOS and Windows.
- The help center highlights desktop extensions, a curated extensions directory, and support for local MCP-server workflows.
- The same article also frames Claude Cowork as a visual interface for Claude Code-like agentic work.
- Cowork can access local files, coordinate parallel workstreams, and run code in an isolated virtual machine with controlled file and network access.
