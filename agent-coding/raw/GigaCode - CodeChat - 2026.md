# GigaCode CodeChat

## Captured Text

- CodeChat is GigaCode's IDE-integrated chat client for natural-language coding help.
- The product can generate code, modify existing project code, explain code, generate tests, generate documentation, and review code.
- The docs expose short commands such as `/explain`, `/doc`, `/test`, `/review`, and `/improve`.
- The interaction model is selection-aware and designed for working on the open file or a highlighted code fragment.
