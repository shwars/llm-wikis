# Hermes Agent skills system docs snapshot

## Captured Text

- Hermes describes skills as on-demand knowledge documents that load progressively to minimize token cost.
- The docs make every installed skill available as a slash command and explain skill hubs, registries, trust levels, and external directories.
- Hermes also distinguishes builtin, official, trusted, and community skill sources with different policy levels.

## Capture Notes

- This source is central for the idea that skills are progressive-disclosure memory rather than always-loaded prompt text.
- It also adds a concrete trust and registry model for coding-agent extensions.
