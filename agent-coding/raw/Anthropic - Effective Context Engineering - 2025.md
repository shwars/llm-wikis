# Effective context engineering for AI agents

## Captured Text

- Anthropic argues that context is a finite resource with diminishing returns and that prompt engineering is giving way to context engineering.
- The essay recommends the smallest high-signal context, hybrid up-front plus just-in-time retrieval, and long-horizon strategies like compaction, note-taking, and multi-agent decomposition.
- It also warns about context rot as sessions grow.

## Capture Notes

- This is a central theory source for LASD context management.
- It helps justify repo-local knowledge stores and separate agent contexts.
