# Hermes Agent overview readme snapshot

## Captured Text

- Hermes Agent describes itself as a self-improving agent that lives on your server and gets more capable over time.
- Its readme emphasizes a closed learning loop, persistent memory, autonomous skill creation, messaging platforms, schedules, subagents, and multiple terminal backends.
- The readme also explicitly positions Hermes as a bridge between personal assistance, coding, research, and training-data generation.
