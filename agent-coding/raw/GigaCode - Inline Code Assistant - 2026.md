# GigaCode Inline Code Assistant

## Captured Text

- GigaCode offers inline AI completions in VS Code and JetBrains IDEs.
- The official docs describe whole-line suggestions, full-function suggestions, and comment-to-code suggestions.
- The product explicitly says the inline assistant is not for high-level design or broad code questions; for that, users should switch to CodeChat.
- GigaCode supports both automatic and manual completion modes as well as multiple suggestion choices.
