# Understanding MCP servers

## Captured Text

- The official server concepts page defines tools, resources, and prompts as the three core MCP server building blocks.
- The page uses filesystem, database, GitHub, Slack, and calendar servers as common examples.
- The docs also distinguish model-controlled tools from application-controlled resources.

## Capture Notes

- This source is central for explaining what a server contributes and how control differs across primitives.
