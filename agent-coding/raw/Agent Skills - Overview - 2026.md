# Agent Skills overview

## Captured Text

- Agent Skills are described as folders of instructions, scripts, and resources that agents can discover and use more accurately and efficiently.
- The site positions the format as a portable, open way to capture company-, team-, and user-specific context and procedures.
- The core value proposition is build once, reuse across multiple agent products and organizations.

## Capture Notes

- This is the main standards-layer source for the skill concept beyond any single vendor.
- It matters because it shows skills as procedural portability and knowledge packaging rather than only product-specific UX.
