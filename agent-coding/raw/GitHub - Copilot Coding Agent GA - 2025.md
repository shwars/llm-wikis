# Copilot coding agent is now generally available

## Captured Text

- GitHub describes Copilot coding agent as an asynchronous autonomous developer agent.
- It works in the background, opens a draft pull request, and uses GitHub Actions-powered development environments.
- The user can delegate tasks by assigning issues or invoking the agent from GitHub or VS Code.
