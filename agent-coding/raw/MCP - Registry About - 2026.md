# The MCP Registry

## Captured Text

- The MCP Registry is the official centralized metadata repository for publicly accessible MCP servers.
- The docs emphasize discovery, standardized metadata, execution instructions, and namespace management.
- The registry is described as an ecosystem service backed by major contributors rather than a single vendor catalog.
