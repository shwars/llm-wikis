# OpenCrust overview readme snapshot

## Captured Text

- OpenCrust describes itself as a secure, lightweight open-source AI agent framework built in Rust.
- Its readme explicitly compares itself against OpenClaw-style systems on memory use, startup time, credential handling, scheduling, routing, and MCP support.
- Security posture, encrypted credential storage, and localhost-first defaults are core product claims.

## Capture Notes

- This source is useful as the security- and efficiency-first fork of the always-on personal-agent family.
- It adds a strong counterpoint to heavier Node-based control-plane designs.
