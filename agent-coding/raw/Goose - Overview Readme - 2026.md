# Goose overview readme snapshot

## Captured Text

- Goose describes itself as a native open-source AI agent for code, workflows, and everything in between.
- Its positioning spans desktop app, CLI, API, multiple model providers, and many MCP-based extensions.
- The readme explicitly says Goose is not only for code but also for research, writing, automation, and data analysis.

## Capture Notes

- This source anchors the desktop/local general-purpose cluster rather than the always-on messaging cluster.
- It matters because it is one of the clearest examples of a local agent that straddles coding and non-coding work without becoming a personal messaging assistant.
