# Hermes Agent MCP docs snapshot

## Captured Text

- Hermes presents MCP as the cleanest adapter layer when a tool already exists outside Hermes.
- The docs cover local stdio and remote HTTP servers, server filtering, dynamic tool discovery, reload behavior, and exposure control.
- Hermes can also run as an MCP server itself so other agents can use Hermes messaging and channel tools.

## Capture Notes

- This source is especially useful because it treats MCP as both an outbound integration layer and an inbound bridge surface.
- It also gives practical security guidance about filtering and least exposure.
