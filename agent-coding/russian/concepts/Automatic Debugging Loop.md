# Automatic debugging loop

## Кратко

Итеративный цикл implement, execute, observe, patch и rerun до тех пор, пока явная проверка не пройдет или неопределенность не будет эскалирована.

## Текущий синтез

Этот цикл — операционное сердце зрелого AI-Assisted Software Development. Модель не просто выдает код; она проверяет код на реальности. Чем дешевле система может исполнять и инспектировать failures, тем ближе agentic coding к автономному построению программного обеспечения.
