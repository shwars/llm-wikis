# Idea file

## Кратко

Структурированный артефакт описания проблемы, предназначенный для повторного использования агентами, а не только для чтения людьми.

## Текущий синтез

Idea file находится между prompt, spec и project brief. Он фиксирует intended artifact на уровне, достаточно абстрактном, чтобы быть переносимым между моделями и инструментами, но достаточно конкретном, чтобы позволить агенту развернуть качественную первую версию без повторного открытия проблемы с нуля.
