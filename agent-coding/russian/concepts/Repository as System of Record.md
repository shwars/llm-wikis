# Репозиторий как system of record

## Кратко

Паттерн хранения инструкций, спецификаций, планов и reference-материалов прямо в репозитории, чтобы агенты могли надежно их использовать.

## Текущий синтез

Репозиторий больше не только место, где живет код; это место, где живет долговечный операционный контекст агента. AGENTS.md, CLAUDE.md, DESIGN.md, references и структурированные docs позволяют репозиторию выступать общей памятью и поверхностью supervision и для людей, и для агентов.
