# AI-Assisted Software Development

## Кратко

Зонтичный термин для использования AI-моделей при создании, изменении, тестировании, понимании и сопровождении ПО.

## Текущий синтез

AI-Assisted Software Development шире, чем chat-based coding help, и шире, чем vibe coding. Он охватывает весь спектр — от completion и pair programming до автономных background agents и repository-shaped harnesses. Поле лучше всего понимать через компоненты вроде problem specification, context management, agent management и verification loops.
