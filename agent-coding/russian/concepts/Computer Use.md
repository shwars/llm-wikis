# Computer use

## Кратко

Способность агента видеть, кликать, печатать и навигировать по графическим интерфейсам как часть выполнения задачи.

## Текущий синтез

Computer use расширяет AI-Assisted Software Development за пределы source files и shell. Как только агент может управлять браузерами, приложениями и desktop-workflows, создание ПО начинает пересекаться с QA, deployment, research и operations. Это увеличивает рычаг, но одновременно повышает требования к safety, supervision и reliability. Именно здесь становится видна конвергенция между coding agents и general-use agents: desktop generalists и always-on personal agents все чаще нуждаются в одних и тех же циклах click, read, type и verification.
