# GitHub

## Кратко

Платформа, которая помогла сделать массовыми AI pair programming, а затем и background coding agents внутри repo workflows.

## Текущий синтез

Траектория GitHub в AI-Assisted Software Development идет от inline pair programming с Copilot к асинхронным coding agents и terminal-native Copilot CLI. Их отличительное преимущество — прямое встраивание делегирования и review в репозиторий и pull-request workflow.
