# OpenAI

## Кратко

Создатель ChatGPT и Codex, а также один из главных источников паттернов agent harness в AI-Assisted Software Development.

## Текущий синтез

Роль OpenAI в этой вики охватывает несколько фаз: разговорный coding через ChatGPT, облачные и локальные coding agents через Codex и явный поворот к harness engineering, где структура репозитория и automations становятся first-class engineering surfaces.
