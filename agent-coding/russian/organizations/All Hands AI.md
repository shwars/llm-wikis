# All Hands AI

## Кратко

Open-source организация, стоящая за OpenHands и связанной инфраструктурой agent development.

## Текущий синтез

All Hands AI заметна тем, что оформляет AI-driven development как открытую платформу, а не как закрытый продукт. OpenHands дает командам еще один путь в AI-Assisted Software Development через self-hosting, SDK, custom sandboxes и community-driven iteration.
