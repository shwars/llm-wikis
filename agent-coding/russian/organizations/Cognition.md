# Cognition

## Кратко

Создатель Devin и заметная компания в категории high-autonomy software agents.

## Текущий синтез

Cognition занимает delegated-engineer конец рынка. Фрейминг Devin помог нормализовать идею о том, что software agent может получать тикеты и фоновую работу, а не быть только интерактивным помощником.
