# Cursor agent and background-agent docs snapshot

## Сводка

Cursor находится на IDE-heavy конце спектра AI-Assisted Software Development: это полированная среда редактирования с автономными agent modes, интерфейсами review и background execution. Именно Cursor помог нормализовать сдвиг от autocomplete к agent mode внутри редактора, а не только в терминале или облаке.

## Ключевые тезисы

- IDE-native agent mode стал устойчивой категорией.
- Background execution может сосуществовать с сильным interactive editor workflow.
- Переключение между ask, edit и autonomous behavior — важный UX pattern.

## Практики и уроки

- Используйте IDE agents, когда важнее всего visual diff review и inline editing.
- Разделяйте planning-, asking- и acting-modes.
- Используйте background agents для асинхронных более крупных задач.
