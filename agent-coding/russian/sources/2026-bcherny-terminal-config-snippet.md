# Boris Cherny on terminal configuration for Claude Code

## Сводка

Тред Черни про настройку терминала напоминает, что agentic coding воплощается в реальных инструментах. Уведомления, multiline input, интеграция с редактором и быстрые механики handoff меняют то, будет ли terminal agent ощущаться плавным или раздражающим в течение многих часов работы.

## Ключевые тезисы

- Мелкие UX- и terminal-детали существенно влияют на throughput агента.
- Качество prompting частично зависит от ergonomics ввода.
- Human supervision loops выигрывают от отзывчивой terminal-feedback.

## Практики и уроки

- Инвестируйте в настройку терминала, уведомления и multiline input.
- Рассматривайте локальную ergonomics как часть harness.
- Снижайте трение вокруг supervision и interruptibility.
