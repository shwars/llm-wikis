# Karpathy on VS Code Cursor + Sonnet 3.5

## Сводка

Этот сниппет фиксирует важный переходный момент: Карпати пишет, что Cursor плюс Sonnet перешли порог, после которого управление на естественном языке и review результата стали эмпирически лучше прежней coding assistance. Центр тяжести сместился от набора синтаксиса к заданию намерения и проверке diffs.

## Ключевые тезисы

- Промптинг плюс review стали net-positive workflow даже для сильного программиста.
- Постановка задач на естественном языке начала заменять заметную долю ручного ввода кода.
- Review и editing остались обязательными.

## Практики и уроки

- Рассматривайте естественный язык как первичную поверхность программирования.
- Переносите усилие с набора текста на review, verification и course correction.
- Используйте агентные инструменты там, где inspection дешев, а итерация быстра.
