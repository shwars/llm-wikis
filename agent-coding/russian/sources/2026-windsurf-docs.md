# Windsurf docs snapshot

## Сводка

Windsurf упаковывает многие уже знакомые AI-Assisted Software Development-примитивы в IDE-first environment: agent chat, memories, rules, worktrees, workflows, AGENTS.md и deploy hooks. Это хороший пример того, как быстро идеи terminal agents были реабсорбированы обратно в графические IDE-продукты.

## Карта заметки

- `[[#Сводка]]`: что это за источник и почему он важен для AI-Assisted Software Development.
- `[[#Ключевые тезисы]]`: главные исторические, продуктовые или теоретические тезисы.
- `[[#Практики и уроки]]`: переиспользуемые операционные уроки.
- `[[#Связанные страницы]]`: какие концепты, инструменты и темы обновляет источник.

## Ключевые тезисы

- IDE-продукты теперь открывают многие из тех же примитивов, что и terminal agents.
- Memory, rules, AGENTS.md и worktrees становятся общим словарем.
- Deployment и workflow automation перемещаются прямо в поверхность редактора.

## Практики и уроки

- Используйте IDE-centric tools, когда принятие командой зависит от привычных GUI affordances.
- Держите memories и rules явными и reviewable.
- Рассматривайте deploy workflows как часть agent harness.
