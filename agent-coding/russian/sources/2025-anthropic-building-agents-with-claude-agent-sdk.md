# Building agents with the Claude Agent SDK

## Сводка

Пост Anthropic про Agent SDK делает важный концептуальный шаг: полезный объект — это не просто coding model, а переиспользуемый agent harness. Один и тот же цикл может обслуживать deep research, note-taking и другие формы knowledge work, расширяя AI-Assisted Software Development от 'AI пишет код' до 'агенты, устроенные как программное обеспечение, выполняют техническую работу'.

## Ключевые тезисы

- Паттерн harness обобщается далеко за пределы кодинга.
- Агентные циклы выигрывают от переиспользуемых skills, memory и интерфейсов supervision.
- Кодинговые агенты — это ранний пример более широкого класса software-shaped agents.

## Практики и уроки

- Проектируйте agent harnesses как переиспользуемую инфраструктуру, а не как одноразовые промпты.
- Переносите стабильные workflows в skills и структурированные инструменты.
- Ищите соседние не-кодинговые задачи с тем же shape цикла.
