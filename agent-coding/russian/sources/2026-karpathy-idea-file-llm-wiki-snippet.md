# Karpathy on the idea file and LLM Wiki

## Сводка

Паттерн idea file продвигает AI-Assisted Software Development еще на шаг дальше: вместо распределения только software или prompts вы распространяете структурированное описание проблемы, ограничений, reference-материалов и intended workflow, чтобы агенты могли многократно разворачивать артефакт. Это паттерн design-document для эпохи агентов.

## Ключевые тезисы

- Артефакты намерения могут быть более переиспользуемыми, чем конкретное сгенерированное приложение.
- Idea files и LLM wikis — долговечные coordination surfaces для агентов.
- Структура управления знанием становится частью создания ПО.

## Практики и уроки

- Пишите переносимые problem files с goals, constraints, references и quality bars.
- Предпочитайте долговечные markdown-артефакты эфемерной prompt-lore.
- Рассматривайте архитектуру знания как часть инженерной архитектуры.
