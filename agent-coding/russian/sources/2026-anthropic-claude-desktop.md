# Install Claude Desktop

## Сводка

Документация Anthropic показывает, что [[russian/tools/Claude Desktop|Claude Desktop]] больше не является просто chat-wrapper. Благодаря extensions, локальному [[russian/concepts/Model Context Protocol|MCP]] и режиму Cowork он становится визуальной surface для agentic work, пересекающейся с coding, research и desktop automation.

## Ключевые тезисы

- Claude Desktop поддерживает локальные extensions и curated discovery этих расширений.
- Cowork переносит Claude Code-like multi-step execution в визуальный desktop workflow.
- Parallel workstreams и isolated execution теперь входят в desktop product-story.

## Практики и уроки

- Используйте desktop agents там, где визуальная supervision важнее shell-native composability.
- Рассматривайте extensions и MCP access как security boundaries, требующие review.
- Используйте Cowork-подобный режим для long-running tasks, которым полезна видимая review-surface.
