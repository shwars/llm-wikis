# Karpathy on LLM Knowledge Bases

## Сводка

Пост Карпати про knowledge bases переосмысляет агентный рычаг вокруг структурированных markdown-систем знания. Тот же ход, который помогает software agents, помогает и research agents: положить raw-материал в стабильный репозиторий, позволить агенту компилировать и поддерживать структурированные заметки и задавать вопросы уже поддерживаемому корпусу, а не перечитывать мир заново каждый раз.

## Ключевые тезисы

- Структурированные markdown-knowledge stores могут превосходить ad hoc retrieval на умеренном масштабе.
- Index files и summaries — first-class context artifacts.
- Agentные паттерны переносятся из codebases на research-corpora.
