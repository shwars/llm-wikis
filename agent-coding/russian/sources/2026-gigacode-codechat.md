# GigaCode CodeChat

## Сводка

Официальная документация CodeChat показывает, что [[russian/tools/GigaCode|GigaCode]] является не только autocomplete-инструментом. Он также поддерживает prompt-review-edit workflows прямо в IDE через chat, объяснение кода, генерацию тестов, генерацию документации, review-команды и трансформацию кода.

## Ключевые тезисы

- CodeChat работает с открытым файлом или выделенными фрагментами кода.
- Short commands превращают частые coding-tasks в повторяемые IDE-workflows.
- GigaCode охватывает больше одного maturity stage, потому что совмещает completion и conversational editing.

## Практики и уроки

- Используйте selection-aware chat для explanation, review и точечных transformations.
- Упаковывайте повторяемые операции в slash-commands, когда командам нужна предсказуемая помощь в IDE.
- Отличайте chat-workflows от чистого completion при классификации coding tools.
