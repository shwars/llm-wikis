# Introducing the Codex app

## Сводка

Codex app делает саму координацию product surface: threads, projects, worktrees, skills и automations становятся first-class сущностями. AI-Assisted Software Development здесь уже не только про генерацию патчей, а про supervision команд агентов во времени — и в кодинге, и в соседних knowledge-work задачах.

## Ключевые тезисы

- Координации нескольких агентов нужен отдельный UX, а не только лучшая модель.
- Skills и automations расширяют coding agents на более широкий круг workflows.
- Worktrees — базовая инфраструктура параллелизма.

## Практики и уроки

- Используйте app- или platform-level coordination для длинной параллельной работы.
- Упаковывайте повторяющиеся workflows в skills и периодические automations.
- Делегируйте не-кодинговые maintenance tasks в тот же agent stack.
