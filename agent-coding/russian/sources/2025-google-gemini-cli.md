# Gemini CLI: your open-source AI agent

## Сводка

Gemini CLI описывает terminal agent как общую локальную утилиту, а не как помощника только для кода. Сочетание open source, scriptability, web grounding, MCP и большого контекста делает его важным и для AI-Assisted Software Development, и для соседних research- или operations-задач.

## Ключевые тезисы

- Terminal agents могут объединять кодинговые и не-кодинговые workflows.
- Открытость и scriptability — ключевые дифференциаторы CLI tools.
- Размер контекста и tool use стали product-level selling points.

## Практики и уроки

- Используйте CLI agents для смешанных automation-, scripting- и research-flows.
- Используйте non-interactive mode для повторяемых задач.
- Подключайте MCP и search grounding, когда важна свежесть информации.
