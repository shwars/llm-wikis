# Introducing GitHub Copilot: your AI pair programmer

## Сводка

GitHub представил Copilot как AI pair programmer, который читает локальный контекст кода и предлагает строки и функции прямо в редакторе. В истории AI-Assisted Software Development это эпоха code completion: человек по-прежнему декомпозирует задачу и отвечает за проверку, а модель ускоряет реализацию у курсора.

## Ключевые тезисы

- Ближайший код и комментарии стали рабочим prompt-контекстом для генерации кода.
- Тесты и исследование API уже тогда проявились как ценные сценарии.
- Модель взаимодействия оставалась assistive, а не автономной.

## Практики и уроки

- Поддерживайте чистый локальный контекст кода: имена и комментарии питают модель.
- Относитесь к подсказкам как к ускорителю, а не как к доказательству корректности.
- Используйте completion-инструменты для узкой реализации и вспоминания паттернов.
