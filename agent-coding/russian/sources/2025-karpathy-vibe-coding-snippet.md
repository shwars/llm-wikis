# Karpathy on vibe coding

## Сводка

Сниппет про vibe coding фиксирует момент, когда разрозненные привычки AI-assisted разработки получили имя. Это отчасти мем, но он описывает и реальный workflow: задавать результат разговорно, позволять модели активно генерировать и вмешиваться в основном через промпты, проверки и вкус.

## Ключевые тезисы

- Vibe coding — это культурная точка входа в более широкий AI-Assisted Software Development, а не вся область.
- Стиль обменивает близость к коду на скорость, итерацию и рычаг.
- Лучше всего он работает для прототипирования, исследования и дешевой ошибки.

## Практики и уроки

- Используйте vibe coding для исследования, а не как замену verification-heavy engineering.
- Ожидайте долгов сопровождения, если человек так и не восстановит модель системы.
- Сочетайте быстрый промптинг с последующим hardening и архитектурной работой.
