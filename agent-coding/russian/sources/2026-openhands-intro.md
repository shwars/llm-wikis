# OpenHands introduction snapshot

## Сводка

OpenHands представляет платформенный open-source конец AI-Assisted Software Development: не просто coding assistant, а software-agent stack с cloud, CLI, SDK, MCP и поддержкой custom sandbox. Особенно важен для команд, которые хотят self-host или глубоко кастомизировать agent harness.

- `[[#Практики и уроки]]`: переиспользуемые операционные уроки.

## Ключевые тезисы

- Open-source команды строят уже полноценные agent platforms, а не только одиночные инструменты.
- SDK и контроль sandbox важны для продвинутого деплоя.
- Customization и self-hosting остаются важной частью ландшафта.

## Практики и уроки

- Используйте platform tools, когда нужны custom sandboxes и integrations.
- Упаковывайте repo setup в scripts и microagents.
- Рассматривайте open-source agent stacks как инфраструктурный выбор, а не только UI-выбор.
