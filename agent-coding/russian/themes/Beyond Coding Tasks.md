# За пределами кодинга

## Кратко

Эта тема отслеживает, где harnesses для coding agents уже используются в соседних технических и knowledge-work loops.

## Текущий синтез

Крупный сдвиг 2025-2026 годов состоит в том, что те же инструменты и harnesses, которые используются для кодинга, теперь применяются для deep research, documentation, note-taking, issue triage, release briefs, CI summaries, visual QA, spreadsheet work и inbox-like follow-up. Общий знаменатель здесь не сам код как output, а способность поддерживать контекст, вызывать tools, проверять результат и переносить работу во времени. Рост general-use agents вроде OpenClaw, Hermes Agent, Memoh и Goose делает это особенно явным: coding превращается в один сильный workflow внутри более широкого agent stack, а не в единственную цель.
