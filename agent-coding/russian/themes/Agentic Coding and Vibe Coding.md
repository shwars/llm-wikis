# Агентное программирование и vibe coding

## Кратко

Эта тема различает широкое поле AI-Assisted Software Development и более узкий мемный стиль, который называется vibe coding.

## Текущий синтез

Agentic coding — широкий зонтичный термин: это системы, которые могут исследовать, планировать, редактировать, запускать команды, проверять результат и иногда работать асинхронно. Vibe coding — более узкий культурный стиль внутри этого зонта, сфокусированный на быстром прототипировании на естественном языке, высоком доверии к генерации и более слабом прямом чтении кода. Практическое различие здесь не между серьезным и несерьезным, а между глубиной verification и ожидаемой стоимостью сопровождения.
