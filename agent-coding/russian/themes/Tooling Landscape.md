# Tooling Landscape

## Кратко

Эта тема суммирует главные продуктовые категории и конкурентные направления в AI-Assisted Software Development на 22 апреля 2026 года.

## Текущий синтез

Рынок теперь охватывает как минимум семь пересекающихся кластеров. [[russian/tools/GitHub Copilot|Code-completion tools]] сохраняют полный контроль за человеком. [[russian/tools/Aider|Prompt-review-edit tools]] и [[russian/tools/Gemini Code Assist|IDE copilots]] добавляют conversational editing и reviewable transformations. [[russian/tools/Claude Code|Terminal agents]] вроде [[russian/tools/Claude Code|Claude Code]], [[russian/tools/OpenCode|OpenCode]], [[russian/tools/Qwen Code|Qwen Code]] и [[russian/tools/Gemini CLI|Gemini CLI]] делают ставку на scriptability и repo-local control. [[russian/tools/Cursor|IDE agents]] вроде [[russian/tools/Cursor|Cursor]] и [[russian/tools/Windsurf|Windsurf]] возвращают эти циклы обратно в редактор. [[russian/tools/Codex|Repo и background agents]] вроде [[russian/tools/Codex|Codex]], [[russian/tools/OpenHands|OpenHands]], [[russian/tools/Devin|Devin]] и [[russian/tools/GitHub Copilot Coding Agent|GitHub Copilot Coding Agent]] сдвигают supervision в сторону tasks, environments и PRs. Наконец, [[russian/tools/Stitch|design-side agents]] и [[russian/tools/Claude Design|artifact-generation tools]] все сильнее влияют на то, что implementation agents делают дальше.

## Карта кластеров

- `Code completion`: [[russian/tools/GitHub Copilot|GitHub Copilot]] и inline-сторона [[russian/tools/GigaCode|GigaCode]].
- `Prompt-review-edit`: [[russian/tools/Aider|Aider]], [[russian/tools/Gemini Code Assist|Gemini Code Assist]] и chat-сторона [[russian/tools/GigaCode|GigaCode]].
- `CLI agentic development`: [[russian/tools/Claude Code|Claude Code]], [[russian/tools/Gemini CLI|Gemini CLI]], [[russian/tools/OpenCode|OpenCode]], [[russian/tools/Qwen Code|Qwen Code]] и [[russian/tools/GitHub Copilot CLI|GitHub Copilot CLI]].
- `IDE agentic development`: [[russian/tools/Cursor|Cursor]], [[russian/tools/Windsurf|Windsurf]] и части [[russian/tools/Antigravity|Antigravity]].
- `Repo или background delegation`: [[russian/tools/Codex|Codex]], [[russian/tools/OpenHands|OpenHands]], [[russian/tools/Devin|Devin]] и [[russian/tools/GitHub Copilot Coding Agent|GitHub Copilot Coding Agent]].
- `Design-adjacent workflow tools`: [[russian/tools/Stitch|Stitch]] и [[russian/tools/Claude Design|Claude Design]].
- `General-use overlap`: [[russian/tools/Claude Desktop|Claude Desktop]] плюс более широкий кластер, описанный в [[russian/analyses/General Use Agents - 2026|General Use Agents - 2026]].
