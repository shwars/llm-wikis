# GitHub Copilot

## Кратко

Иконический IDE pair-programmer, определивший современную эпоху code completion в AI-assisted software development.

## Текущий синтез

[[russian/tools/GitHub Copilot|GitHub Copilot]] — самый ясный anchor для стадии code completion в исторической модели этой вики. Даже когда экосистема GitHub расширилась в сторону CLI и background agents, исходный продукт остался главным ориентиром для inline assistive coding.

## Профиль инструмента

- `Primary surface`: IDE.
- `Maturity stage`: Code completion.
- `Autonomy model`: low-autonomy assistive completion и chat.
- `Strengths`: низкий порог, сильная inline ergonomics, историческое влияние на поле.
- `Limits`: намного слабее в task delegation, чем более поздние CLI и repo-native agents.
