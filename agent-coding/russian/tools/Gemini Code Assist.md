# Gemini Code Assist

## Кратко

IDE-first coding assistant от Google, который сочетает code completion, chat, citations и движение в сторону agent mode.

## Текущий синтез

[[russian/tools/Gemini Code Assist|Gemini Code Assist]] показывает, как современные IDE assistants все чаще объединяют три слоя: completion, conversational editing и путь к agent mode. Его лучше понимать как мост Google между классическими IDE copilots и более автономными terminal или platform agents.

## Профиль инструмента

- `Primary surface`: IDE.
- `Maturity stage`: Prompt-review-edit workflow.
- `Autonomy model`: assistive to semi-agentic, с документированным движением к agent mode.
- `Strengths`: IDE integration, chat, citations и низкопороговое внедрение для команд, уже живущих в editors.
- `Limits`: по умолчанию менее автономен, чем terminal-first или repo-first agent systems.
