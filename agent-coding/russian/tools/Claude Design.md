# Claude Design

## Кратко

Design-adjacent agentic workspace для прототипов, слайдов и визуальных артефактов, питающих software workflows.

## Текущий синтез

[[russian/tools/Claude Design|Claude Design]] не является code editor, но должен быть в этом каталоге, потому что превращает visual intent в machine-usable artifacts, способные направлять implementation. В этой вики он находится на границе design generation и подготовки coding workflow.

## Профиль инструмента

- `Primary surface`: visual workspace.
- `Maturity stage`: Prompt-review-edit workflow.
- `Autonomy model`: визуальная генерация и iterative refinement с human review.
- `Strengths`: rapid prototyping, artifact generation и design-to-implementation handoff.
- `Limits`: хуже подходит для прямых repo operations, чем coding-native agents.
