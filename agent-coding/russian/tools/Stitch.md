# Stitch

## Кратко

AI-native design canvas от Google для high-fidelity UI generation, iteration и handoff в software workflows.

## Текущий синтез

[[russian/tools/Stitch|Stitch]] должен быть в этом каталоге, потому что превращает design intent в editable artifacts и code-adjacent outputs, которые затем направляют implementation. Это скорее design-side control surface для AI-assisted software development, чем direct coding tool.

## Профиль инструмента

- `Primary surface`: visual workspace.
- `Maturity stage`: Agentic development.
- `Autonomy model`: agentic design exploration с iterative human review.
- `Strengths`: быстрая UI-идеация, multimodal prompting и ценность design-to-code handoff.
- `Limits`: не заменяет repo-operating coding agents.
