# Memoh

## Кратко

Container-first, multi-bot платформа always-on agents, которая объединяет persistent memory, channels, skills и MCP.

## Текущий синтез

Memoh сдвигает паттерн general-use agents в сторону multi-bot operations и admin infrastructure. Он важен тем, что выглядит уже не как chatbot, а как hosted agent platform с isolation, memory, channels, skills и MCP под одной крышей. Это сближает его с задачами серьезной эксплуатации coding agents, хотя продукт не является coding-first.

## Профиль инструмента

- `Primary surface`: web UI, channels, containerized bot-runtime и admin-config.
- `Target use case`: always-on multi-bot automation с persistent context и явной изоляцией.
- `Autonomy model`: long-running containerized agents с доступом к channels и tools.
- `Context and memory`: persistent memory и per-bot filesystem- или environment-boundaries.
- `Extensibility`: MCP, skills, slash commands и настройка models или channels.
- `Security and ops`: isolation и graphical administration входят в core value proposition.
- `Coding relationship`: хорошая модель для команд, которым нужна agent-infrastructure вокруг coding-adjacent work, а не только редакторный assistant.
