# Gemini CLI

## Кратко

Terminal agent от Google, охватывающий coding, research и automation при помощи MCP и web-grounded workflows.

## Текущий синтез

[[russian/tools/Gemini CLI|Gemini CLI]] важен тем, что трактует terminal agent как general local utility, а не как coding-only assistant. Он находится в том же семействе, что [[russian/tools/Claude Code|Claude Code]] и [[russian/tools/OpenCode|OpenCode]], но сильнее связан с более широким Google ecosystem.

## Профиль инструмента

- `Primary surface`: CLI / terminal.
- `Maturity stage`: Agentic development.
- `Autonomy model`: интерактивный terminal agent с tool use и scriptability.
- `Strengths`: open-source posture, web grounding, MCP и гибкая utility за пределами coding.
- `Limits`: более слабые repo-native delegation surfaces, чем у worktree-first или PR-first systems.
