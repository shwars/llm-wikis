# OpenCode

## Кратко

Open-source terminal coding agent, который делает ставку на privacy, provider choice и repo-local instructions.

## Текущий синтез

[[russian/tools/OpenCode|OpenCode]] важен как доказательство того, что Claude Code / Codex workflow уже реплицируется в open ecosystem. Он усиливает мысль о том, что repo-local instructions и terminal-first agent loops становятся стандартными интерфейсами, а не single-vendor peculiarities.

## Профиль инструмента

- `Primary surface`: CLI / terminal.
- `Maturity stage`: Agentic development.
- `Autonomy model`: интерактивный coding agent с multi-session и provider-flexible workflows.
- `Strengths`: open source, inspectability, privacy-first posture и AGENTS.md-friendly flow.
- `Limits`: в некоторых supervision surfaces полировка ниже, чем у флагманских closed products.
