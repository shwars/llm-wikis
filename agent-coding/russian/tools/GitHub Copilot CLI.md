# GitHub Copilot CLI

## Кратко

Terminal-native companion от GitHub, переносящий Copilot-style assistance в command-line workflows.

## Текущий синтез

[[russian/tools/GitHub Copilot CLI|GitHub Copilot CLI]] показывает, как completion-era бренды переупаковываются в terminal agents. Он ближе к интерактивной terminal assistance, чем к полностью delegated repo execution, но ясно выводит семейство Copilot за пределы editor-only help.

## Профиль инструмента

- `Primary surface`: CLI / terminal.
- `Maturity stage`: Agentic development.
- `Autonomy model`: интерактивный terminal agent внутри GitHub ecosystem.
- `Strengths`: GitHub integration, близость к shell и низкий порог для команд, уже использующих Copilot.
- `Limits`: менее orchestration-heavy, чем специализированные background-agent platforms.
