# Antigravity

## Кратко

Agent-first development platform от Google, построенная вокруг editor, terminal, browser и manager-style orchestration surfaces.

## Текущий синтез

[[russian/tools/Antigravity|Antigravity]] представляет собой самый ясный Google-native переход от IDE assistance к orchestration нескольких агентов и verifiable artifacts. Для этой вики он отмечает переход от editor help к platform-level supervision.

## Профиль инструмента

- `Primary surface`: IDE / platform.
- `Maturity stage`: Multi-agent development.
- `Autonomy model`: high-autonomy agents с manager-style orchestration.
- `Strengths`: agent-first UX, cross-surface execution и artifact-based trust building.
- `Limits`: платформа сложнее, чем простые editor copilots или CLI tools.
