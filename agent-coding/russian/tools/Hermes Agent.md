# Hermes Agent

## Кратко

Самосовершенствующийся persistent general-use agent, который объединяет memory, skills, schedules, messaging и coding-capable work.

## Текущий синтез

Hermes Agent находится на пересечении persistent personal assistance и coding-capable agent infrastructure. Его особенность в том, что memory и skill creation становятся частью самого agent-loop. Поэтому для этой вики Hermes особенно важен: это один из самых ясных примеров того, как идеи coding agents расширяются в long-lived general-use system.

## Профиль инструмента

- `Primary surface`: CLI, messaging-gateway, scheduled work и server-hosted sessions.
- `Target use case`: persistent personal- или team-agent workflows для coding, research и automation.
- `Autonomy model`: long-running agent с background work, schedules и delegation через subagents.
- `Context and memory`: cross-session memory, reuse skills и self-improvement loops.
- `Extensibility`: native skills, hubs и registries, режим MCP-client и режим MCP-server.
- `Security and ops`: сильная configurability и exposure-control, но нужна дисциплинированная эксплуатация.
- `Coding relationship`: ближе к coding, чем OpenClaw, потому что прямо поддерживает subagents, coding-skills и tool composition.
