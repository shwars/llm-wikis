# Goose

## Кратко

Локальный desktop/CLI general-purpose agent, который охватывает coding, research, writing, automation и data-work.

## Текущий синтез

Goose — самый наглядный пограничный кейс в этом расширении: локальный general-purpose agent, который уверенно включает coding, но не определяется только им. Он важен тем, что показывает, как MCP и local agent-surfaces позволяют одному продукту охватывать software-work, research, writing и automation, не превращаясь в always-on personal assistant.

## Профиль инструмента

- `Primary surface`: native desktop app, CLI и API.
- `Target use case`: local general-purpose work для code, research, automation, writing и data-analysis.
- `Autonomy model`: интерактивный локальный агент, а не messaging-native always-on assistant.
- `Context and memory`: local runtime-context с extension-driven доступом к внешним системам.
- `Extensibility`: сильный акцент на MCP плюс multi-provider support.
- `Security and ops`: local-first deployment и open-source extensibility — важные selling points.
- `Coding relationship`: связь прямая и сильная, но coding подается как один из нескольких главных workflows.
