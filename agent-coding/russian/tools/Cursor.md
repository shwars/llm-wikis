# Cursor

## Кратко

IDE-first agentic coding environment, который помог нормализовать natural-language editing и background-agent workflows.

## Текущий синтез

[[russian/tools/Cursor|Cursor]] — один из самых ясных мостов от conversational editing к полному agent mode внутри редактора. Исторически он важен потому, что помог перевести команды от prompt-review-edit привычек к более автономной разработке под supervision IDE.

## Профиль инструмента

- `Primary surface`: IDE.
- `Maturity stage`: Agentic development.
- `Autonomy model`: editor-native agent execution с optional background work.
- `Strengths`: плотный editing-loop, discoverable UI и сильная diff-and-review ergonomics.
- `Limits`: по-прежнему более editor-centric, чем repo-native или orchestration-first systems.
