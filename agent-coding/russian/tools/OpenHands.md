# OpenHands

## Кратко

Open-source платформа для agent-driven software development через cloud, CLI, SDK и custom sandbox surfaces.

## Текущий синтез

[[russian/tools/OpenHands|OpenHands]] важен тем, что упаковывает agentic development как platform, а не как assistant. Он особенно релевантен командам, которым нужны self-hosting, custom sandboxes, SDK control или более глубокая интеграция с собственным harness.

## Профиль инструмента

- `Primary surface`: repo / cloud / SDK.
- `Maturity stage`: Multi-agent development.
- `Autonomy model`: high-autonomy delegated development на extensible platform.
- `Strengths`: open-source stack, self-hosting, SDK access и customizable execution environments.
- `Limits`: операционный overhead выше, чем у turnkey IDE или CLI products.
